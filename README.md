## Usage
1. Create .env file with this content:
```
SECRET_KEY=randomlettersandnumbers123
```
2. To start \n
    Windows:
    ```bash
    docker-compose up --build 
    ```
    Linux:
    ```bash
    docker compose up --build 
    ```
3. To stop \n
    Windows:
    ```bash
    docker-compose down -v
    ```
    Linux:
    ```bash
    docker compose down -v
    ```
