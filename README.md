## Usage
1. Edit and rename .env.example file to .env

2. To start
    Windows:
    ```bash
    docker-compose up --build 
    ```
    Linux:
    ```bash
    docker compose up --build 
    ```
3. To stop
    Windows:
    ```bash
    docker-compose down -v
    ```
    Linux:
    ```bash
    docker compose down -v
    ```
